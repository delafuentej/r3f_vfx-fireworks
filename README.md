# 🎆 Sky Adventure – Fireworks Show

**Sky Adventure** is an interactive 3D experience where we launch fireworks from a floating island in space. We use modern technologies like **React Three Fiber**, **Three.js**, **Tailwind CSS**, and a custom **VFX engine** to create a spectacular visualization with particles, lights, sounds, and dynamic camera movement.

---

## 🚀 Features

- 🌍 Floating 3D island as the base scene.
- ✨ Particle system for fireworks powered by our custom `wawa-vfx` engine.
- 🌈 Visual effects like bloom and vibrant colors.
- ☁️ Custom clouds and dynamic sky.
- 🔊 Sound effects synchronized with fireworks.
- 🎮 UI with buttons to launch different types of fireworks.
- 📦 State management with `zustand`.
- 🧪 Real-time parameter control using `leva`.

---

## 🧱 Technologies Used

- **React Three Fiber** – React renderer for 3D graphics.
- **Three.js** – Core 3D graphics library.
- **@react-three/drei** – Useful 3D helpers and components.
- **@react-three/postprocessing** – Postprocessing effects (e.g., bloom).
- **Tailwind CSS** – Utility-first CSS framework.
- **Leva** – GUI for live parameter tweaking.
- **Zustand** – Lightweight global state management.
- **vite-plugin-glsl** – Support for custom GLSL shaders.
